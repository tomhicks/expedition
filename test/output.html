• Feature: Verify charts

    • Scenario: US7 - Ability to compare spending in a certain category with the previous months

      • Given I am an authenticated user

      • And I go to the menu item "Transactions"

      • When I change the amount spent in category Gas for the current month

      • And I change the amount spent in category Gas for the previous month

      • Then The chart for the current month is updated with the amount "90.00" for "Gas"

      • And The chart for the previous month is updated with the amount "190.00" for "Gas"

    • Scenario: US8 - Ability to add new transaction and verify chart is plotted correctly

      • Given I am an authenticated user

      • And I go to the menu item "Transactions"

      • When I add a new transaction with category "Telephone" and amount "35" for the current month

      • And I add a new transaction with category "Telephone" and amount "38" for the previous month

      • Then The chart for the current month is updated with the amount "35.00" for "Telephone"

      • And The chart for the previous month is updated with the amount "38.00" for "Telephone"
